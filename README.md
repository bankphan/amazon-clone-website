# Amazon Clone Website

This project is a static clone of the Amazon website homepage, built to demonstrate front-end development skills.

## Features

- **Responsive Navigation Bar**: Includes location, search, language, and account sections.
- **Hero Image Slider**: A functional image carousel for featured content.
- **Product Grids and Sliders**: Displays product categories and best sellers in an organized layout.
- **Footer**: Replicates the standard Amazon footer links and copyright information.

## Technologies Used

- **HTML5**: For the structural foundation of the web pages.
- **CSS3**: For styling, layout (Flexbox/Grid), and responsiveness.
- **JavaScript**: For interactive elements like the image slider.

## Setup Instructions

1.  Clone the repository or download the source code.
2.  Navigate to the project directory.
3.  Open `index.html` in your preferred web browser to view the website.
